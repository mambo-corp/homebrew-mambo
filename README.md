# Homebrew tap for Mambo

[Mambo](https://trymambo.app/) is a keyboard-driven, multi-engine, zero-telemetry desktop database client for macOS.

## Install

```bash
brew install --cask mambo-corp/mambo/mambo
```

Or in two steps:

```bash
brew tap mambo-corp/mambo
brew install --cask mambo
```

## Update

```bash
brew upgrade --cask mambo
```

Mambo also has a built-in updater that prompts you in-app when a new release is available — both paths work.

## Uninstall

```bash
brew uninstall --cask mambo
```

To also remove preferences and caches:

```bash
brew uninstall --cask --zap mambo
```

## Requirements

- macOS 11 (Big Sur) or newer
- Apple Silicon or Intel
